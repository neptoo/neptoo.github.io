技术相关

这些是我遇到过、实际使用过、想要尝试的的技术

Frontend

  • 目前擅长的程式语言是 JavaScript
  • 喜欢 UI 还有前端相关的新技术
  • 最常使用的前端框架是 Vue
    • 喜欢的技术栈组合（前端）是 Vue + vuex + Element UI + Axios+ flex，可以用来处理各种复杂的场合
    • 若涉及的状态不复杂，会使用 store 模式
    • 如果是小中型项目，会直接用 vuex
  • 使用过微信小程序 + Vant 对公司项目进行维护和开发，可以熟练使用
  • 有用 tailwind 开发，便携的、易上手的、拯救选择困难症的！
  • 最近想要专注于
    • React / Canvas / Three.js / Webgl
    • 信息可视化 / storytelling
    • 使用者互动、用户交互

Backend / 其他

  • Node.js
  • 大学中主修的语言是 Java，有机会再去学习 Swift
  • 数据库：会写 SQL 语法
