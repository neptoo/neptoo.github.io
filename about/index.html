About

2018-07-11
3 minute read

简介

常用 neptoo 這個英文昵称，因为取名字是很麻烦的事情。

现居武汉，19年毕业于信管专业，从事前端工作两年半，現在在找工作。

因為我很不太擅长自我介紹，所以我把我想到的事情全部記錄在這裡，以後有人要我自我介紹我就可以把這個網站丟給她，然後看完後大概就知道我是怎樣的人。

以下看完大約需要 3 ~ 5 分鐘。

  • Github

我的个性

  • 真诚：会直接的表达自己的感受，比如夸赞、喜爱、焦虑、害怕。
  • 慢热，有时在思考对方的话，可能回应的比较慢。
  • 可以主动和初次见面的人聊天，但取決于对方的回应。
  • 容易紧张。
  • 热衷于帮助别人：分享自己的经验，帮助别人解决问题。
  • 遇到难题先分解：列出已知的必要条件，再慢慢补充，比一下子想到解决方法要容易很多。
  • 基本上不挑食，会尽量吃完，除非食物本身不新鲜、或者太难吃。

价值观

  • 认为只要他人不危害社会健康，选择过怎样的生活都应该被尊重，Embrace Diversity。
  • 认为可以健康成长、接受教育、身体没有残缺或者特殊疾病已经是很幸运的事情。

雷点

  • 歧视女性、歧视少数群体、bodyshaming
  • 急忙否定別人，扣帽子，不愿意认真听别人的话
  • 承上，不尊重或者干涉他人的选择

兴趣

  • 影：纪录片/POI
  • 饮：坚持每天至少喝4杯水，经常喝的有茉莉花茶、菊花茶，山楂。
  • 吃：偶尔做饭，偶尔堂食，禁外卖。
  • 音乐：英伦摇滚为主，也喜欢indie pop，目前喜欢的乐队/乐手有：
    • Pulp, Glass Animals
    • Florence + The Machine
    • Halsey, Frank Ocean，St. Vincent
  • 书籍：不限类型，只要有趣就会想要看看
    • 《女性主义有什么用》
    • 《遥远的向日葵地》、《羊道三部曲》-李娟，散文
    • 诗集、杂文

职业目标

在自己喜欢的领域成为自信的人。

想利用自己会的技能帮助到别人，以下的事情都会是我想做的事：

  • 用好看的方式呈现有意思的东西
  • 写一些小工具解决生活中的问题（比如地铁禁品清单查询）
  • 给自己喜欢的库提交贡献
  • 开发一个有意思好玩有用的APP
  • 在 WIT(women in tech) 里互帮互助，和其他女性从业者一起进步，给予想要从事这个行业的女生一些帮助

喜欢的频道

  • JavaScript30 推荐给所有自学者的 JS 课程，非常赞！
  • Traversy Media : 北美阮一峰/FreeCodeCamp二代， 涉猎广，活跃，WEB开发和编程相关教程。

技术相关

這些是我遇到过、实际使用过、想要尝试的的技术。

Frontend

  • 目前擅長的程式語言是 JavaScript
  • 喜歡 UI 還有前端（瀏覽器）相關的新技術
  • 最常使用的前端框架是 Vue
    • 喜欢的技术栈组合（前端）是 Vue + vuex + Element UI + Axios+ flex，可以用来处理各种复杂的场合
    • 若涉及的状态不复杂，会使用 store 模式
    • 如果是小中型项目，会直接用 vuex
  • 使用过微信小程序 + Vant 对公司项目进行维护和开发，可以熟练使用
  • 有用 tailwind 开发官网，是便携的、易上手的、拯救选择困难症的工具
  • 比較沒有在特別的領域深耕，因此最近想要專注於
    • 信息可视化 / storytelling
    • 使用者互动、用户交互
    • 多媒体
    • 前端在 AI 的应用

Backend / 其他

  • 大学中主修的语言是 Java，有机会再去复习 iOS 开发以及 Swift
  • 数据库：会写 SQL 语法。

结尾

這大概是我能告訴你關於我的一切，希望我們可以找到共同話題。
